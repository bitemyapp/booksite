When to expect a release

We have done a release once every month since May 2015 in the middle of the month. We plan to continue to do so.

Available in early access right now:

  • Front matter
  • Lambda Calculus
  • Hello Haskell
  • Strings
  • Basic Datatypes
  • Types
  • Typeclasses
  • Functions
  • Recursion
  • Lists
  • Folds
  • Algebraic datatypes
  • Signaling Adversity
    Covers how to use Maybe, Either, and kinds in more detail.
  • Building projects in Haskell
    Covers modules, basics of Cabal, IO, and do syntax.
  • Testing
    Covers the basics of using Hspec and QuickCheck to write tests.
  • Monoid, Semigroup
  • Functors
  • Applicative
  • Monad (SPOOKY? No.)

On deck (>50% complete):

  • Foldable
  • Traversable
  • Reader
  • State

Still being written:

  • Parsers
  • Monad Transformers
  • Non-strictness and efficiency
  • Commonly used data structures
  • (Four final chapters TBA)

Length of the book

All page counts are in terms of the screen-friendly rendering you'll see in the sample PDF. The currently available version of the book is 797 pages in the PDF, with 771 numbered pages. We expect the final version of the book to be over 900 pages long in the PDF.
