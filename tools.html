--- title: Suggested tools when learning Haskell ---

Getting Haskell installed

You'll need to get Haskell installed before you can do anything. One of the authors has written a guide which includes install instructions. Instructions for Mac and Windows users are provided as well.

Recommended

  • Don't know what text editor to use? We recommend Sublime Text or Atom
  • We nd some of our readers that use OS X have tested Haskell for Mac and found it exceptional as a combined text editor and workspace for learning Haskell.
